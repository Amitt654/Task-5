# 🌳 Decision Tree & Random Forest Classification

## 📌 Project Overview

This project demonstrates the implementation of **Decision Tree** and **Random Forest** algorithms for binary classification using the **Heart Disease Dataset**. It compares the performance of both models, analyzes overfitting, visualizes the decision tree, evaluates feature importance, and validates the model using cross-validation.

---

## 🎯 Objectives

* Import and preprocess the dataset
* Split the data into training and testing sets
* Train a Decision Tree Classifier
* Visualize the Decision Tree
* Analyze overfitting using training and testing accuracy
* Control overfitting by limiting tree depth
* Train a Random Forest Classifier
* Compare the accuracy of both models
* Interpret feature importance
* Evaluate the model using 5-fold Cross-Validation

---
